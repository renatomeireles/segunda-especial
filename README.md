## Compatibility

## Installing
Install globally:

    npm install -g segunda-especial

## Running
Run command:

    segundaespecial

Or import in NodeJS script:

    import segundaespecial from 'segundaespecial';

    segundaespecial();
    
## Running with [npx](https://www.npmjs.com/package/npx)
Run command:

    npx segunda-especial
